## 重写 TDSQL_DESIGN.md 设计文档

**背景**：用户要求基于腾讯云官方文档 (https://cloud.tencent.com/document/product/557/8767) 为 watchmen-storage-tdsql 包生成/更新设计文档。

**关键发现**：
- 原有设计文档存在 DDL 语法错误：使用了 `SHARDING_COLUMN = xxx, SHARDING_TYPE = HASH`，这不是合法的 TDSQL SQL 语法
- 正确的 TDSQL DDL 语法（来自官方文档）：
  - HASH 分表：`CREATE TABLE ... ) shardkey=col ENGINE=...`
  - RANGE 分表：`CREATE TABLE ... ) TDSQL_DISTRIBUTED BY RANGE(col) (s1 VALUES LESS THAN(...)) ENGINE=...`
  - LIST 分表：`CREATE TABLE ... ) TDSQL_DISTRIBUTED BY LIST(col) (s1 VALUES IN(...)) ENGINE=...`
